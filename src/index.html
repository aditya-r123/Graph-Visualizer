How to Use

Add Vertex: Left-click anywhere on canvas
Move Vertex: Drag vertices around
Create Edge: Right-click two vertices consecutively
Custom Labels: Set label in sidebar before adding vertex (all labels must be unique)
Edit Labels: Left-click any vertex to edit its label (Enter to save, Escape to cancel)
Edit Mode: Long-press (2 seconds) on any vertex or edge to enter edit mode
Custom Styling: Use edit mode to customize colors, fonts, and sizes for individual elements
Search: Enter target vertex and run BFS/DFS
Save/Load: Save graphs with unique names and switch between them safely
